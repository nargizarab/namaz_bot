# 🕌 Намаз-бот для Telegram

Бот для напоминания и учёта намазов по шиитскому ладу для города **Алматы**.  
Использует API **Aladhan** (метод Тегеранского университета).

---

## 📦 Установка

```bash
# 1. Создай виртуальное окружение
python -m venv venv
source venv/bin/activate   # Linux/Mac
venv\Scripts\activate      # Windows

# 2. Установи зависимости
pip install -r requirements.txt
```

---

## ⚙️ Настройка

### 1. Получи токен бота
- Открой [@BotFather](https://t.me/BotFather) в Telegram
- Напиши `/newbot`, следуй инструкциям
- Скопируй токен

### 2. Пропиши токен

**Вариант A — переменная окружения (рекомендуется):**
```bash
export BOT_TOKEN="1234567890:ABC-ваш-токен"
```

**Вариант B — в файле `config.py`:**
```python
BOT_TOKEN = "1234567890:ABC-ваш-токен"
```

## 🚀 Запуск

```bash
python bot.py
```

## 🕐 Логика уведомлений

### Блоки намазов (шиитский лад)

| Блок | Намазы | Напоминание | Опросник |
|------|--------|-------------|----------|
| Блок 1 | 🌙 Фаджр | За 5 мин до Фаджра | В момент восхода солнца |
| Блок 2 | ☀️ Зухр + Аср | За 5 мин до Асра | В момент начала Магриба |
| Блок 3 | 🌆 Магриб + Иша | За 5 мин до Иша | Через 90 мин после Иша |

### Недельная сводка
- Каждое **воскресенье** после окончания Иша бот присылает итоги недели.

---

## 📋 Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Запуск и регистрация |
| `/times` | Времена намазов на сегодня |
| `/stats` | Статистика текущей недели |
| `/week` | Итоги прошлой недели |

## 📁 Структура проекта

```
namaz_bot/
├── bot.py          # Точка входа
├── config.py       # Конфигурация (токен, город, метод)
├── prayer_api.py   # Получение времён намазов (Aladhan API)
├── database.py     # Хранение данных (JSON-файл)
├── handlers.py     # Обработчики команд и кнопок
├── scheduler.py    # Планировщик напоминаний
├── requirements.txt
└── README.md
```

---

## 🗄 Хранение данных

Данные хранятся в файле `namaz_data.json` (создаётся автоматически).  
Для продакшена можно заменить на SQLite/PostgreSQL — нужно лишь переписать `database.py`.

## 🌐 API

- **Aladhan API:** https://aladhan.com/prayer-times-api  
- **Метод:** 13 (University of Tehran)  
- **Координаты Алматы:** 43.2220°N, 76.8512°E
